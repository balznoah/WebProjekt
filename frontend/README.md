# Frontend

Das Frontend ist eine einfache HTML-Datei – kein Build-Schritt nötig!

## Option A: Direkt im Browser öffnen (einfachste Methode)
Öffne `src/index.html` direkt im Browser per Doppelklick.

## Option B: Mit lokalem Webserver (falls CORS-Probleme)
```bash
npm start
# Öffnet http://localhost:4200
```

## HINWEIS
- Kein `ng serve`, kein Angular CLI, kein Build nötig
- Das Backend läuft auf http://localhost:3000 (API Gateway)
- start-local.sh im Hauptordner muss laufen!
