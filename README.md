## 🚀 Starten

### 1. Codespace öffnen
Auf GitHub den grünen **"Code"** Button klicken → **"Codespaces"** → **"Create codespace on main"**

Warten bis alles durchgestartet ist (~2 Min).

### 2. Ports öffentlich schalten

Im Terminal:

```bash
gh codespace ports visibility 3000:public 4200:public
```

### 3. App öffnen

Unten auf den Tab **"Ports"** klicken → neben Port **4200** auf den Link klicken.
